# ani-ui
Animate component for vue.js 2.0

## Getting Started
### using `npm` or `yarn`
```shell script
npm install ani-ui@latest
```
```shell script
yarn add ani-ui@latest
```

### using CDN
```html
<link rel="stylesheet" href="https://unpkg.com/ani-ui/dist/ani-ui.css">
<script src="https://unpkg.com/ani-ui/dist/ani-ui.common.js"/>
```
